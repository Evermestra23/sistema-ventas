¡Bienvenido a TENLIO!

Este es un sistema de archivo local. Tus datos se guardan en este navegador, pero para abrirlos en otro navegador o computador necesitas usar el archivo de sesión guardado.

💡 Cómo funciona: Cada vez que uses el sistema, presiona "💾 Guardar Sesión" en la barra superior. Esto descarga un archivo .tenlio con todos tus datos. La próxima vez que abras TENLIO en cualquier navegador, carga ese archivo y tendrás todo tu historial listo.
